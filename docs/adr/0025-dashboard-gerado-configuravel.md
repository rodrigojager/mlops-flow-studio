# Dashboard gerado configurável

Decidimos que o runtime autônomo deve incluir um dashboard operacional próprio, gerado a partir dos blocos, métricas e contratos do projeto. No MVP, o Studio deve gerar páginas padrão e permitir configuração visual simples, como ativar páginas, escolher métricas destacadas, definir rótulos e selecionar o modelo/pipeline padrão para teste.
