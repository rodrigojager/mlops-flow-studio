# Preview seguro de input/output

Decidimos que o Studio deve mostrar previews resumidos de input/output por padrão, com schema, shape, amostras, estatísticas e links para artefatos, em vez de renderizar datasets completos no inspector. Isso mantém a UI responsiva, reduz exposição acidental de dados sensíveis e ainda permite inspeção completa por artefato, download, amostra expandida ou viewer dedicado.
