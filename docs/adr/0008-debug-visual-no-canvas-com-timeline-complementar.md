# Debug visual no canvas com timeline complementar

Decidimos que o acompanhamento de execução deve aparecer primeiro no canvas, com estado visual por nó, aresta e bloco composto. A timeline hierárquica é complementar para navegação temporal e investigação, mas não substitui a leitura direta do grafo, porque o usuário precisa reconhecer rapidamente onde o pipeline está, o que já rodou, o que falhou e por onde os dados passaram.
